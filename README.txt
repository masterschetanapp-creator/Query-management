Lead Tracker Shared Offline App

Keep this complete folder on your shared drive.

Open Start_LeadTracker.bat from every PC.

The shared database is saved as leadtracker_db.json in this same folder.
The initial database was created from Lead_Tracker.xlsx.
Automatic backups are saved in db_backups.

Do not open index.html directly for shared database mode.
Direct opening works only as single-computer browser storage mode.
